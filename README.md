该项目为[12306购票](https://kyfw.12306.cn/otn/leftTicket/init)的命令行形式。目前实现的功能有:
- 余票查询 (Query_Ticket)
- 购票 (Order_Ticket)
- 取消订单 (Cancel_Order)
- 退票 (Refund_Ticket)
****

Version: Python3
